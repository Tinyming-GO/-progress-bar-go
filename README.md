## 安装

```shell
go get -u github.com/Tinyming-GO/progress-bar-go
```
## 如何使用
```go
package main

import (
	"time"

	progress "github.com/Tinyming-GO/progress-bar-go"
)

func main() {
	bar := progress.New(100)
	for i := 0; i < 100; i++ {
		time.Sleep(time.Second / 10)
		bar.Done(1)
	}
	bar.Finish()
}
```
